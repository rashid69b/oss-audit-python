\# OSS Audit Python Project



\## Description

This project contains basic shell scripts to perform simple system checks and Python-related tasks.



\## Features

\- Create a file using script

\- Install Python automatically

\- List installed Python packages

\- Show system information

\- Check Python version



\## Requirements

\- Linux Operating System

\- Python 3 installed



\## How to Run

1\. Open terminal

2\. Navigate to scripts folder:

&#x20;  cd scripts

3\. Run any script using:

&#x20;  bash filename.sh



Example:

bash create\_file.sh
